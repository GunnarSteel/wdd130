Because of Video games my memory has improved so much. Over the past six years I have noticed that my memory retention has improved so much that I have been able to remeber things that I should have forgotten a long time ago. Each day since I have noticed that my not just my memory has improved but a lot om my brains cognitive abilities have improved allowing me to be able to learn and apply things in my life faster and more efficiently.

Video games aren’t just a way for entertainment, they also possess the capability to expand our brain’s memory retention. Some of these capabilities allow us to do our day-to-day jobs better. The ways video games help with memory retention are with problem-solving and puzzles. These memory builders are only just a few that help those who play video games today. Every video game has its own strong point that it works on. Each one of those points helps with our brain’s memory and increases the capacity of our memory. There are so many more ways that video games help us with memory retention that I might add on later.
